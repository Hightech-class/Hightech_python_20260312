### 1. 파이썬 주요 특징
* **인터프리터 언어(스크립트 언어)** : 코드의 일부만을 실행시킬 수 있는 언어
* **동적 타입 언어** : 하나의 변수가 여러 가지 자료유형을 다룰 수 있는 언어(유연성)
* **플랫폼 독립적 언어** : 내가 작성한 프로그램이 OS의 영향을 받지 않음. 대부분 운영체계에서 모두 동작함(윈도우, 리눅스, 맥 등)

### 2. 파이썬 코드 작성 시 유의사항
* 파이썬 프로그래밍 시 소문자, 대문자, 숫자, `_`(underscore)로 조합 가능 (**대소문자 구분함**)
    * `_`(underscore)는 남발하지 않고 최대한 자제해서 쓰기 (프로그래밍에 혼란을 줄 수도 있음)
* **숫자로 시작해서는 안 됨** (위반: `1var` / 가능: `var1`)
* **키워드**(파이썬 문법의 식별자)를 식별자로 사용 X (예: `Global = 10`, `if = 5`)
    * *식별자: 무언가를 다른 것과 구분해서 부를 수 있는 변수*
* **특정 심벌 사용 X** (`!`, `@`, `#`, `$`, `%` 등)
* **식별자 길이 상관 X** (추천 방식: 단어 앞자리마다 대문자 섞기 / 예: `ThisIsALongVariable`)
* **한 줄에 하나의 명령문만 해석**
    * 길어질 경우 괄호, 역슬래시(`\`) 등으로 처리
    * 예시:
      ```python
      a = (1+2+3+
           4+5+6+
           7+8+9)
      ```

### 3. 파이썬 Identation, Comments
* **들여쓰기(Identation)**
    * 파이썬에서는 들여쓰기로 코드의 블록을 정의함
    * 일반적으로 4칸을 띄움 (tab 키 사용)
* **주석(Comments)**
    * 해쉬(hash) 기호 `#`으로 구성 (`'''` 또는 `"""` 기호로 다중 주석 처리)
    * *참고: 큰따옴표(`"""`)
