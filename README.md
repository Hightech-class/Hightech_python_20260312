* 파이썬 데이터 유형(Python data types)(자동차예시)
  ** number(정수: 메모리 가용내 길이제한없음 / 실수: 소수점이하 15자리까지 정확(소수점 자리가 부정확할수도있다))
  ** list
    *** item(리터널, 자료구조(리터널 모음)의 순서가 지정된 목록(list)
    *** list의 item이 다른 유형이 가능함
    *** 선언 : [](대괄호) 내부에 쉼표로 구분하여 나열
    *** item의 인덱스(index)는 0부터 시작! # <- 매우 중요함
      **** ex1) a[2] -> 3번째를 출력
      **** ex2) a[0:3] -> 0번째부터 3보다작은숫자(2번째)차례까지 출력 -> 0~2번째까지 출력
      **** ex3) a[5:] -> 5번째부터 쭉 출력
  ** tuple
    *** list와 기능은 동일, 선언 이후에는 변경x(일반적으로, list보다 속도가 빠름)
    *** 선언 : 괄호() 내부에 쉼표로 구분하여 아이템을 나열함
  ** string
    *** list, tuple처럼 슬라이싱 연산자 []를 사용하여 접근
      **** ex1) s = 'Hello world' <- * hello, world사이 가운데 공백도 문자로 취급함
  ** set
    *** 외부 데이터를 가져올때 주로 사용함
    *** 고유한(unique)아이템들의 정렬되지 않은 모음(중복되어 있지x)
    *** 선언 : 중괄호 {} 내부에 쉼표로 구분하여 아이템 나열
    *** slicing operation을 통해 접근불가 -> [] (순서를 모르니 접근x)
    *** 변경 가능한 아이템(list 등)은 가질수x
  ** dictionary
    *** key-value 쌍(pair)의 정렬되지 않은 모음
    *** 데이터의 양이 많을떄 주로 사용함(key를 통해 value 검색)
    *** 선언 : 중괄호 {} 내부에 쉼표로 구분하여 key:value 나열
  ** 형변환(conversion and type casting)
    *** 암묵적컨셉(implicit) : 파이썬 인터프리터가 자동으로 형변환 실시,사용자 개입 필요x
    *** 명시적컨셉(Explicit) : 사용자가 원하는 형태로 변경(int(), float(), str()등)

* 파이썬 input, output, import
  ** output
    *** print() : 화면으로 데이터를 출력하기 위한 함수(function)
    *** str.format() : 화면 출력 표현방식 편의 제공, 문자열&입력 객체 정보 조합으로 구성
  ** input : 사용자가 데이터를 입력받도록 지원(여기서 받는건 다 문자로 처리함)
  ** import : 모듈을 가져와서 쓸수있게 해줌

* 파이썬 연산자(Python operators)
  ** 산술(arithmetic)연산자 : +, -, *, /, %, //, **
  ** 비교(comparison)연산자 : >, <, ==, !=, >=, <=
  ** 논리(logical)연산자 : and, or, not
  ** 비트(bitwise)연산자(요즘은 잘안쓰긴함) : >>(곱하기), <<(나누기)
  ** 대입(assignment)연산자 : ex) a += 5 -> a = a + 5(첨엔 정석대로 쓰다가 익숙해지면 대입연산자 쓰기)
  ** 항등(identity)연산자 : 두개의 값이 메모리의 동일한 부분에 있는지 확인하는데 사용함
   *** 리터널 모음 : (ex. a= 10 , b = [10,11] ) 리스트([]) 자체는 생성할 때마다 새로운 주소를 가지지만, 그 안에 담긴 리터럴(숫자 10 등)은 기존에 존재하는 값의 주소를 재사용하여 가리킴
                    
* Name(identifier) : 객체에 부여된 이름
  ** 파이썬의 모든건 객체, 기본개체에 접근하기위한 방법

* Namespace : 변수의 적용 범위를 구분하여 사용
